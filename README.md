# NAME

Plagger::Plugin::CustomFeed::GoogleSearchHistory - Custom feed for Google Search History

# SYNOPSIS

    - module: CustomFeed::GoogleSearchHistory
      config:
        username: username 
        password: p4ssw0rd

# DESCRIPTION

This plugin fetches your Google Search History log. 

# CONFIG

- username, password

    Your Google ID and password.
